Dictionry

Set

Swift 的标准库中，Set 和 OptionSet 实现了 SetAlgebra 协议。

Foundation 中， IndexSet 和 CharacterSet 也实现了 SetAlgebra 协议。

  • IndexSet

    IndexSet 是一个正整数的集合。其内部使用了一组范围列表进行实现，比 Set 更高效。

  • CharacterSet

    CharacterSet 是一个高效存储 Unicode 编码的集合。
